# flutter_custom_cards

New Flutter package which helps developers to create beautiful custom cards.

## Getting Started

To import the widget:

```dart
import 'package:flutter_custom_cards/flutter_custom_card.dart'
```

Make a widget *Card*:

```dart
CustomCard(
text:"Hello World",
color:Colors.red,
)
```

#### Example

```dart
CustomCard(
text:"Hello World",
color:Colors.red,
)
